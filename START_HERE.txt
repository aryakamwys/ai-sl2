üéØ LANGKAH INSTALASI CEPAT (5 MENIT)

1. INSTALL DEPENDENCIES
   ----------------------
   Buka PowerShell/Terminal di folder ini, lalu:
   
   pip install -r requirements.txt

2. SETUP API KEY GEMINI
   ---------------------
   a. Buka: https://makersuite.google.com/app/apikey
   b. Login dengan Google Account
   c. Klik "Create API Key"
   d. Copy API key yang muncul
   e. Buka file: .streamlit/secrets.toml
   f. Ganti YOUR_GEMINI_API_KEY_HERE dengan API key Anda

3. JALANKAN APLIKASI
   ------------------
   streamlit run main.py

4. BUKA BROWSER
   ------------
   Aplikasi otomatis terbuka di: http://localhost:8501
   Atau buka manual jika tidak otomatis
